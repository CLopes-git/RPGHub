# RPGHub

## Sobre o projeto

O **RPGHub** é uma aplicação web desenvolvida para facilitar a criação, organização e gerenciamento de fichas de personagens de RPG.

A plataforma permite que jogadores criem, visualizem e editem seus personagens de forma prática, centralizando todas as informações em um único ambiente digital.

O sistema busca substituir fichas físicas e planilhas desorganizadas, oferecendo uma experiência mais intuitiva, moderna e acessível para jogadores de RPG.
